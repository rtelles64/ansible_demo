# Ansible Demo

This project just demonstrates a CICD Pipeline using Ansible,
CloudFormation, and Circle CI.

## File Tree

```
/README.md
|
|
/template.yml
|
|
/.circleci/
     |
     |
     + config.yml
```

  - `config.yml` contains the code to handle the CICD pipeline

  - `template.yml` actually creates the instances
